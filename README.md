<h1>Airplane</h1>
<img width="572" alt="image" src="https://user-images.githubusercontent.com/103854287/197001950-a8b88835-7c8f-40e7-95fc-b4d89fffb5ba.png">
목표 : 체력이 다른 Enemy를 죽이면서 점수를 얻으면서 안전하게 본진까지 도착하기
-모든 물체와 충돌이 있으면 죽는다.
-체력이 다른 적을 죽이면서 다른 점수를 얻는다(체력이 높으면 높은점수)
-적이 근처에 오면 우측 하단에 그림이 나오면서 노래가 나온다.
-적을 공격으로 제거하면 Particle로 폭팔을 시킨다.
<준비기간> 3주



<h1>Rocket</h1>
<img width="1111" alt="image" src="https://user-images.githubusercontent.com/103854287/195649550-a7c3f43e-380a-4753-abf4-5afdb3662db8.png">
목표 : 로켓을 이용하여 도착지에 가서 다음 스테이지로 넘어가기 <br/><br/>

-Rigidbody를 이용하여 객체의 물리엔진의 영향을 받도록 하여 현실감 있게 동작<br/>
-Transform.Rotate를 이용하여 회전<br/>
-중력은 달에 간 것처럼 중력 값을 낮춤<br/>
-오디오 : 로켓이 발사 , 충돌 , 목적지에 도착을 하면 오디오가 나온다.<br/>
-충돌시 Particle <br/>
